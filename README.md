# Scrape

A lightweight **web scraping and data extraction tool** for collecting structured data from websites.

## Overview
This project scrapes web pages, parses HTML content, and outputs structured JSON data for downstream use such as analytics, storage, or APIs. It is deployed as a simple web service using Vercel.

## Focus
- HTML parsing and DOM traversal  
- JavaScript-based scraping logic  
- Structuring scraped data into JSON  
- Lightweight backend deployment  

## Tech Stack
- **JavaScript (Node.js)**
- **HTML**
- **JSON**
- **MongoDB**
- **Vercel**

## Deployment
Live deployment available via Vercel.
